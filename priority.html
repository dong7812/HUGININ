P1 — 검색 품질
검색 고도화 — MMR + 하이브리드
현재 pure cosine similarity → 유사 결정 중복 노출 문제
  • 1단계 MMR: 후보 K개에서 관련성 + 다양성 균형 재정렬
    score = 0.7·sim(doc,query) − 0.3·max sim(doc,선택된것)
  • 2단계: BM25 + vector 하이브리드 (pg_textsearch)
    키워드 정밀도 + 시맨틱 이해 동시에
  • 근거: InfoGain-RAG(arxiv 2509.12765) — naive RAG 대비 +17.9%
  • 적용 위치: 서버 search endpoint 후처리, DB 변경 없음
